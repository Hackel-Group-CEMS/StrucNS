# StrucNS
This repository contains the revelant code and datasets for StrucNS.

Folded protein function requires stability, yet mapping protein structure and sequence to a fitness landscape remains difficult. The protein fold is the physical realization of complex, spatially-sensitive physicochemical interactions among residues; quantitatively elucidating how these subtle relationships dictate thermodynamic stability remains challenging. We present StrucNS, a mathematical framework that identifies principles governing protein fitness by employing network science to learn the physicochemical relationships between residues and their contributions to stability directly from the protein fold. Representing the fold as a network topology, we utilize an inverse approach: while the fold is traditionally viewed as the phenotypic consequence of the underlying chemical forces, we demonstrate that this topology can be used to decode the very physicochemical dependencies that govern protein stability. Unlike protein language models reliant on high-dimensional evolutionary embeddings, StrucNS extracts these signals directly from the interaction-weighted network topology. Independence from evolutionary history uniquely suits StrucNS for de novo design prediction. Despite reduced dimensionality and training depth, StrucNS outperforms ESM-2 and ProteinMPNN on predicting mutational stability. StrucNS outperforms supervised UniRep in predicting absolute stability of de novo designs. Feature analysis reveals network topology as the key driver of model predictive power, as it contributes 59\% of model importance; moreover topology feature ablation reduces recall by 53\%. SHAP analysis reveals that two highly influential features are high degree and low modularity of polar/hydrophobic mixed subnetworks, which highlights the importance of connectivity between the hydrophobic core and protein surface. Revelation of these predictive topological features underscores the utility of an interpretable model.
